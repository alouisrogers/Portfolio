Tai Fighter Level 2 Vehicle

The challenge for this project was to build a dual deploy rocket for a Level 2 certification. This project was useful to be able to design a rocket airframe with aerodynamics in mind. A 4:1 Tangent Ogive Nosecone and tailcone were used to reduce subsonic flow separation and a tailcone was fabricated to reduce pressure drag at the aft of the rocket. To create a successful dual deploy system, I wired an electronics bay based on Stratologger alitmeters for deployment events, an Arduino for GPS and data logging, and an XBEE to trasmit the data to a ground station. flight events and give telemetry data using a gps module.

Technologies Used: XBEE GPS, Stratologger Alimeters


Custom Bluetooth Speaker

I wanted a stylish bluetooth speaker with a deep sound that could hit lower end frequencies. This result is difficuly for small bluetooth speakers which struggle with battery and driver size due space constrictions. In order to achieve the intended result, I built a speaker and determined drivers that would give the sound I wanted and a battery that would be able to power it with ample battery life. The bluetooth speaker used a 3.5" subwoofer and 2" full range driver with passive filtering and 1000Hz crossover enclosed in a hardwood case. To get more depth out of the base, a port was 3d printed to let air out. The project resulted in a very nice looking speaker to take to beach days and to friend's houses.


Audio Visualizer Speaker

The goal of this project was to create an all-in-one audio visualizer and speaker to give to my sister as a Christmas present. This was a fun quick build, which used a fully 3d printed housing and two 2" full range drivers for a good range of sound. The visualizer circuit used an arduino micro and code employing fast forrier transforms to separate the incoming audio signal into bands to be displayed on an 8x32 LED matrix, which could switch modes using a built in button. The circuit was all soldered to a perforated prototype board and code was modified from an existing arduino project. In the end, the speakers sounded good, but the visualizer seemed to have low peaks as a result of noise in the signal.

Technologies Used: Arduino


The Perfect Truss

For my final project for my artificial intelligence class, my partner and I made a program that when given a number of beams, made a truss that would be structurally stable under a given load at each connection point of the structure. The problem was simplified using rigid members in two dimensions. The system was modeled as a constraint satisfaction problem accelerated with a heuristic calculated from the normal equations using strength increasing factors from mechanics. A Finite Element solver was used to solve for deflection and stress concentration of the beams to determine if the generated structure was stable or not and were drawn using the turtle graphics library. We were able to get some interesting structures out of the program and lots of triangles which was expected.

Technologies Used: Python, PyCharm


3D Printer Build

I wanted to have 3d printing capability for other projects so building a 3d printer became a project in its own right. I purchased a 3D printer kit from a Chinese company through Ebay. The 3d printer had to be assembled and it broke multiple times and parts needed to be replaced and serviced, such as the hot end assembly. Parts needed to be soldered to the control board to fix shorts and code had to be updated to the most recent version of the open source Marlin 3d printer software. This project resulted in my being able to construct 3d parts for other future projects.

Technologies Used: C++, Arduino, Marlin, Cura

Building a Budget Computer

I made a challenge with a friend to build the cheapest, best gaming computer possible. This required reading the specifications of each part to ensure compatibility and power requirements. I had to obtain components from multiple sources like Facebook, Ebay and local used part stores. The older generation motherboard did not have connectors for the newer case, so voltages had to be tested from the power supply and motherboard to wire and solder the necessary connections to the board and power supply to get things running smoothly. In the end, my friend ended up with a computer that can run AAA games at 60 fps for around 300$.

I used the SDK from the leap motion controller to write a program that would open spotify and play music with the wave of a hand. The leap motion controller was not great at recognizing individual fingers so the funtionality was kept limited.

Technologies Used: Leap Motion, Python, Pycharm

Rocket Fin Analysis

The rocket club needed to redesign the rocket fins for aerodynamics and understand its performance increase over the normal flat edge fins. We used Solidworks Flowsim to run simulations on a custom airfoiled rocket fin for an aerodynamic rocket build. Using batch runs and adjustable parameters, I was able to get a Force vs. Wind Velocity curve that showed the expected quadratic relationship. Using these softwares, I was able to get a design with 50% of the aerodynamic drag of the previous fin.

Technologies Used: Solidworks, Flowsim
